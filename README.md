# Twitter-Sentiment-vs-Bitcoin-Price
An end-to-end machine learning project that analyzes Twitter sentiment and explores its relationship with Bitcoin price movements. The project combines NLP techniques, sentiment analysis, data visualization, and market trend analysis to uncover patterns between social media sentiment and cryptocurrency markets.
